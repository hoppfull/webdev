Import external file (F# script)
This in an example of how to load an F# script module from another F# script. This is actually not recommendend. If code is too large to keep in one script file, consider creating a source project.

mymodule.fsx: 
module whatevs

let greeting = "whatsup"
Now to load that module into main program myprog.fsx: 
#load "mymodule.fsx"

printfn "%s" whatevs.greeting
Run script with command console:
projectfolder\>fsi myprog.fsx whatsup projectfolder\>_
