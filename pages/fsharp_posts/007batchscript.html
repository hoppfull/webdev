Simplifying compilation
Writing down the compilation commands in the console every time we want to compile will be quite unmanageable for even small projects. To deal with this we create batch scripts. Basically a batch script is a text file where we store console commands so that we don't have to write them down manually.

To create a batch script simply create a text file and name it something like "compile.bat". To reduce clutter in the terminal when running I start all scripts with:
@ECHO OFF

We make calls in the command console with the CALL keyword. Example:
CALL cd..
This is the equivalent of:
C:\dir\>cd.. C:\>_

For example: we have a source file we want to compile and then immediately run the result.

myprog.fs: 
printfn "hello batch scripting!!!11!one!"
cmplandrun.bat:
@ECHO OFF CALL fsc myprog.fs CALL myprog.exe
Now we run the script:
projectfolder\>cmplandrun Microsoft (R) F# Compiler Version 12.0.12005.1 Copyright (c) Microsoft Corporation. All Rights Reserved. hello batch scripting!!!11!one! projectfolder\>_
