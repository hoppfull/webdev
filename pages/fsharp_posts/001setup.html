Setting up F# in windows
F# is included in an installation of .Net. Add an environment variable to path of the F# compiler. For me this happen to be: "C:\Program Files (x86)\Microsoft SDKs\F#\3.1\Framework\v4.0\".

We need access to fsi.exe and fsc.exe. fsi.exe gives us access to the REPL and is used to run scripts which is great for experimenting while learning F#. fsc is used to compile source files into .exe-files for distribution.

Try running fsi and fsc in command console like this:
C:\> fsi Microsoft (R) F# Interactive version 12.0.21005.1 Copyright (C) Microsoft Corporation. All Rights Reserved. For help type #help;; > _

You're in the REPL now. To exit type "#quit;;" and press enter:
> #quit;; C:\>_
If you're seeing this then you're good to go. Else go use your Google-muscles.

Next let's see if the F#-compiler works like it should You should see this:
C:\>fsc Microsoft (R) F# Interactive version 12.0.21005.1 Copyright (C) Microsoft Corporation. All Rights Reserved. error FS0207: No input specified C:\>_
Don't worry about the error. We didn't give the compiler any files to compile so all is good.
