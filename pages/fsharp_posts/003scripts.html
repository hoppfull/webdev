F# scripts
Script files are also very good for learning and experimenting. But I'm sure there are legitimate uses in projects for scripts. You don't have to compile scripts, you can just run them straight away.

Create a project folder somewhere. Inside your folder, create a text file and name it something like myscript.fsx. Write some code in myscript.fsx: 
printfn "Hello World!"

Navigate to your project directory in the command console (hint: press shift + right click the project folder and press "Open command window here") and run the script with fsi:
...\projectfolder\>fsi myscript.fsx Hello World! ...\projectfolder\>_
