Import external file (C#)
Functional programming is great for most problems but sometimes you want to optimize snippets of code an imperative language. C# is such a language included with .Net and works great together with F#.

First make sure that you've got an environment variable to the C# compiler csc.exe. For me this is located in "C:\Windows\Microsoft.NET\Framework64\v4.0.30319".

First we create a C# module and name it MyModule.cs: 
namespace MyModule {
  public class MyClass {
    public static string x = "Whats up??!?";
  }
}
Then we compile it with csc.exe into a DLL library:
...\projectfolder\>csc /t:library MyModule.cs Microsoft (R) Visual C# Compiler version 4.0.30319.33440 for Microsoft (R) .Net Framework 4.5 Copyright (C) Microsoft Corporation. All rights reserved. ...\projectfolder\>_
This will output a library file MyModule.dll which can now be used by F# programs.

Now we create an F# program that can use the DLL file, main.fs: 
open MyModule

printfn "%s" MyClass.x
Compile with fsc:
...\projectfolder\>fsc -r:MyModule.dll main.fs Microsoft (R) F# Compiler version 12.0.21005.1 Copyright (c) Microsoft Corporation. All Rights Reserved. ...\projectfolder\>_
This will output an executable main.exe. It will require MyModule.dll to work.

Running main.exe produces following result:
...\projectfolder\>main.exe Whats up??!? ...\projectfolder\>_
