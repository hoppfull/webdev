The REPL
The REPL is used to work with F# interactively. Like for testing small snippets of code which can be very useful during development or learning for experimentation.

The REPL is started in the console by running fsi:
C:\> fsi Microsoft (R) F# Interactive version 12.0.21005.1 Copyright (C) Microsoft Corporation. All Rights Reserved. For help type #help;; > _

We write code and end a statement with ";;":
> printfn "hello whatsup";; hello whatsup val it : unit = () > _

Exit the REPL with the "#quit;;" command:
> #quit;; C:\>_
