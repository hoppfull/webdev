Compiling source manually
I'm not going to bother explaining compiling source with an IDE since there's plenty of info the webz.

Create a project folder and inside it create a text file named something like "myprog.fs".
Add some code: 
printfn "yo!"

Open command console and navigate to your project folder and compile source with fsc:
...\projectfolder\>fsc myprog.fs Microsoft (R) F# Interactive version 12.0.21005.1 Copyright (C) Microsoft Corporation. All Rights Reserved. ...\projectfolder\>_
The compiler will have created an executable myprog.exe in project folder. You should now in theory be able to distribute your program and run it on any computer with .Net installed.

Run your compiled program with the console in order to see its output:
...\projectfolder\>myprog yo! ...\projectfolder\>_
