Import external file (F#)
Sometimes we want to organize our code by putting related code in different files.

A safety feature of F# is that files can only inherit from each other in a one way order. That means file A cannot inherit from file B if B inherits from A. The way this is implemented is through the order in which we compile the files. Parents first followed by their children.

We got two kinds of external files to inherit; namespaces and modules. Modules can contain defined values while namespaces can only contain functions (There's probably more to it).

Create a parent file called mymodule.fs and define it as a module named whatever then define a value: 
module whatever

let x = "hello my friends!"

Now create a child file called myprog.fs and use data from the whatever module: 
printfn "%s" whatever.x

When compiling make sure to list all files in project and order them accordingly:
...\projectfolder\>fsc mymodule.fs myprog.fs Microsoft (R) F# Interactive version 12.0.21005.1 Copyright (C) Microsoft Corporation. All Rights Reserved. ..\projectfolder\>_
Run myprog.exe:
...\projectfolder\>myprog hello my friends! ...\projectfolder\>_
