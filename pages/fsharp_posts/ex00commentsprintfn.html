ex00 - Comments and printfn
Comments are blocks of text in our source files that will not be evaluated at runtime by F#. We insert comments into our code to explain with our own words what is going on.

Ignore text in one line only (called single line comments): 
//this is a single line comment!
Ignore text between two points (called multi line comments): 
(*this is a multi line comment!
yay!*)

printfn is the print function that returns output from your application to the command console. Personally this function makes up 99% of debugging for me when programming.

printfn can only return output from a string directly. No numbers or not even the string content in a variable. But there are tricks to get around that - string formating. 
//This will output "hello" to the command console:
printfn "hello"

//This will not work:
printfn 42
printfn 3.14

//Even the content of a variable will not work:
let x = "whatsup"
printfn x

// To get around this we use string formating:
printfn "%i" 42  //cmd console:42
printfn "%d" 3.14  //cmd console:3.14
printfn "%s" x  //cmd console:whatsup
